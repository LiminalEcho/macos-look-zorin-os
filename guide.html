macOS Look on Zorin OS 17 Core

A complete customization guide for old MacBooks and any Ubuntu-based PC

Tested on MacBook Air 2013 · Zorin OS 17 Core · GNOME 46 · Wayland

Overview

This guide transforms Zorin OS 17 Core into a clean macOS-style desktop. It covers boot screen, login screen, desktop theme, dock, icons, cursors, fonts, sound, touchpad, battery optimization, and security. Everything here uses free, open-source tools unless otherwise noted.

Quick Install

If you want to skip the manual steps, use the automated script:

chmod +x scripts/mac-look-zorin.sh
./scripts/mac-look-zorin.sh

Then follow the manual steps printed at the end of the script output.

Step-by-Step Guide

STEP 01 Apple Plymouth Boot Screen

Replaces the Zorin boot logo with an Apple-style animation.

sudo apt install git
git clone https://github.com/fathyar/mac-plymouth.git && cd mac-plymouth
sudo cp -r mac /usr/share/plymouth/themes
sudo update-alternatives --install \
  /usr/share/plymouth/themes/default.plymouth \
  default.plymouth \
  /usr/share/plymouth/themes/mac/mac.plymouth 100
sudo update-alternatives --config default.plymouth
sudo update-initramfs -u

Preview without rebooting:

sudo plymouthd; sudo plymouth --show-splash; sleep 5; sudo plymouth --quit

Plymouth config (/etc/plymouth/plymouthd.conf):

[Daemon]
Theme=mac
ShowDelay=0
STEP 02 WhiteSur Dark Theme

Installs the macOS Big Sur inspired GTK theme, wallpapers, and GDM login screen.

sudo apt install imagemagick sassc
git clone https://github.com/vinceliuice/WhiteSur-gtk-theme.git
cd WhiteSur-gtk-theme

# Install desktop theme
./install.sh -c Dark

# Install wallpapers
git clone https://github.com/vinceliuice/WhiteSur-wallpapers.git
cd WhiteSur-wallpapers && ./install-gnome-backgrounds.sh && cd ..

# Install GDM login/lock screen
sudo ./tweaks.sh -g

Apply in GNOME Tweaks > Appearance:

SettingValue
ShellWhiteSur-Dark-solid
IconsWhiteSur-light
CursorMcMojave-cursors
Legacy ApplicationsZorinBlue-Dark or WhiteSur-Dark-solid
Light theme users: Replace -c Dark with -c Light and set Shell and Legacy Applications to WhiteSur-Light-solid in GNOME Tweaks.
STEP 03 McMojave Cursors 
git clone https://github.com/yeyushengfan258/McMojave-cursors.git
cd McMojave-cursors && ./install.sh

Then set in GNOME Tweaks > Appearance > Cursor.

STEP 04 WhiteSur Icon Theme 
git clone https://github.com/vinceliuice/WhiteSur-icon-theme.git
cd WhiteSur-icon-theme && ./install.sh

Then set in GNOME Tweaks > Appearance > Icons.

STEP 05 Remove GDM Logo

Step 1: Open the GDM config file:

sudo nano /etc/gdm3/greeter.dconf-defaults

Find the logo line and set it to empty:

logo=''

Save with Ctrl+X, then Y, then Enter.

Step 2: Apply and reset:

sudo dconf update
sudo rm -rf /var/lib/gdm3/.config/dconf
sudo reboot
STEP 06 Fonts 
sudo apt install fonts-inter fonts-jetbrains-mono

Then open GNOME Tweaks > Fonts and set:

FieldValue
Interface TextInter Regular 11
Document TextInter Regular 11
Monospace TextJetBrains Mono Regular 11
Legacy Window TitlesInter Bold 11
AntialiasingSubpixel (for LCD screens)
HintingSlight
STEP 07 macOS Sound Theme 
git clone https://github.com/gxanshu/macos-bigsur-sound-theme-linux.git
cd macos-bigsur-sound-theme-linux
mkdir -p ~/.local/share/sounds
cp -r theme ~/.local/share/sounds/bigsur

Activate it:

gsettings set org.gnome.desktop.sound theme-name 'bigsur'
gsettings set org.gnome.desktop.sound event-sounds true
The sounds are extracted from macOS Big Sur and are technically Apple's property. Use at your own discretion.
STEP 08 Dock and Layout

Open Zorin Appearance from the app menu:

  • Go to Layout and select the bottom dock layout
  • Go to Windows and set titlebar buttons to Left
STEP 09 Touchpad — Mac-like Scrolling 
sudo apt install xserver-xorg-input-libinput
sudo nano /etc/X11/xorg.conf.d/30-touchpad.conf

Add this content:

Section "InputClass"
    Identifier "touchpad"
    Driver "libinput"
    MatchIsTouchpad "on"
    Option "NaturalScrolling" "true"
    Option "ScrollPixelDistance" "60"
    Option "Accel Speed" "0.3"
    Option "Tapping" "on"
    Option "TappingDrag" "on"
    Option "DisableWhileTyping" "on"
EndSection
ScrollPixelDistance of 60 gives the closest feel to macOS. Adjust between 50-75 to taste.
STEP 10 Lid Close — Suspend on Clamshell 
sudo nano /etc/systemd/logind.conf

Set these values:

HandleLidSwitch=suspend
HandleLidSwitchExternalPower=suspend
sudo systemctl restart systemd-logind
STEP 11 Boot Time Optimization 
# Disable NetworkManager wait (~6 seconds saved)
sudo systemctl disable NetworkManager-wait-online.service

# Disable snapd seeded (~1 second saved)
sudo systemctl disable snapd.seeded.service
Keep fwupd.service enabled for firmware security updates.
STEP 12 SSD Optimization 
echo 'vm.swappiness=10' | sudo tee -a /etc/sysctl.conf
sudo sysctl -p
STEP 13 Battery Optimization — TLP 
sudo apt install tlp tlp-rdw
sudo systemctl enable tlp
sudo systemctl start tlp
sudo tlp-stat -s

Check battery health anytime:

upower -i $(upower -e | grep battery)
STEP 14 Fan Control — mbpfan 
sudo apt install mbpfan
sudo systemctl enable mbpfan
sudo systemctl start mbpfan
sudo systemctl status mbpfan
STEP 15 Security — Enable Firewall 
sudo ufw enable
sudo ufw status
STEP 16 System Backup — Timeshift 
sudo apt install timeshift

Open Timeshift, select RSYNC, enable Include Only Hidden Files for both users, then create a snapshot:

sudo timeshift --create --comments "macOS setup complete - $(date +%Y-%m-%d)"
STEP 17 RAM Optimization (4GB Systems) 
# Install ZRAM
sudo apt install zram-config
sudo reboot

# Remove GNOME file indexer
sudo apt remove tracker tracker-miner-fs -y
sudo apt autoremove -y

# Disable GNOME remote desktop
sudo systemctl disable gnome-remote-desktop.service
sudo systemctl stop gnome-remote-desktop.service

Install Tab Suspender in Brave to reduce browser RAM usage from 3.6GB to under 2GB with 10-20 tabs open.

MetricBeforeAfter
RAM used3.1GB1.8GB
RAM available630MB2.0GB
Swap used1.8GB858MB
STEP 18 macOS-style Keyboard Shortcuts

Swap Command and Ctrl so Command handles copy/paste and all window shortcuts:

gsettings set org.gnome.desktop.input-sources xkb-options "['ctrl:swap_lwin_lctl']"
gsettings set org.gnome.mutter overlay-key ''

Log out and back in, then run the shortcuts script:

chmod +x ~/mac-shortcuts.sh
./mac-shortcuts.sh
ShortcutAction
Cmd+C/V/XCopy/Paste/Cut
Cmd+QClose window
Cmd+TabSwitch apps
Cmd+SpaceSearch
Cmd+Shift+3Screenshot
Cmd+Shift+4Screenshot window
Cmd+Shift+5Screenshot tool

Monthly Maintenance

sudo apt update && sudo apt upgrade -y
sudo fwupdmgr refresh && sudo fwupdmgr update
sudo apt autoremove -y && sudo apt clean
sudo journalctl --vacuum-time=7d
rm -rf ~/.local/share/Trash/*
